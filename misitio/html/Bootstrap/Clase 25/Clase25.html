Instalacion local bootstrap

Grids:
Se define en filas y cada fila tiene 12 columnas

Clases: Despues de la raya se pone la cantidad

  • Row
  • col-md-
  • col-
  • col-lg-
  • col-sm-
  • col-x
Areas:
  • Xs(menos de 576)
  • sm ( mayor o igual a 576)
  • md (mayor o igual a 768px)
  • lg (mayor o igual a 992px)
  • xl(mayor o igual a 1200 px)
  • xxl (mayor o igual a 1400px)
Contenedores: se usan el .row para las filas, el .col para columnas y el .container para el contenedor; .container.fluid crea un contenedor responsivo que cubre el 100% del ancho de la ventana
